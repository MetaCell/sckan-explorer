# SCKAN Explorer

Sckanner (Beta) is a web based platform to visualize tool and a key component of the SPARC Program is SCKAN. It is a semantic store housing a comprehensive knowledge base of autonomic nervous system (ANS) and peripheral nervous system (PNS) nerve to end organ connectivity. Connectivity information is derived from SPARC experts, SPARC data, literature and textbooks. SCKAN supports reasoning and offers powerful query and visualization capabilities.

To read more about how to setup the sckanner locally checkout - [Setup README](./applications/sckanner/README.md)
